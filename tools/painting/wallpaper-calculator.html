How to Use This Calculator

Wallpaper estimation is trickier than paint because of pattern matching. Our calculator accounts for pattern repeat.

  1. Measure room dimensions
  2. Check wallpaper roll specs
  3. Enter measurements
  4. Buy all rolls from same dye lot

Frequently Asked Questions

How many rolls of wallpaper do I need?
A standard US double roll (33 ft x 20.5 in) covers about 56 sq ft. For a 12x10 room with 8 ft ceilings, you need approximately 6-8 double rolls.
What is pattern repeat?
Pattern repeat is the vertical distance before the pattern repeats. Large repeats waste more paper for matching.
