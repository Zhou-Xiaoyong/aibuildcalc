How to Use This Calculator

Thin brick veneer count depends on brick size and mortar joint width. Factoring both into the effective module prevents running short mid-wall.

  1. Measure the total wall area in square feet
  2. Enter the brick length and height in inches
  3. Enter the mortar joint thickness (3/8 in is standard)
  4. Order 5% extra for cuts, breaks, and corner pieces
Expert Insights Smart analysis for your Siding project
🧱
Module Math
Adding the joint to both brick length and height gives the effective module. The calculator converts that to bricks per square foot automatically.
🪣
Mortar Sizing
Type S mortar yields roughly 50 thin bricks per 80 lb bag when used for both the setting bed and joint grouting.
✂️
Waste Allowance
A 5% overage covers cuts at openings and corners. Buy full boxes from one lot to keep color consistent across the wall.

Calculation Formula

Eff W = Brick Length + Joint Eff H = Brick Height + Joint Per sq ft = 144 / (Eff W x Eff H) Bricks = ceil(Wall Area x Per sq ft x 1.05) Mortar Bags = ceil(Bricks / 50) Cost = ceil(Bricks x $0.80)

Frequently Asked Questions

How many thin bricks do I need per square foot?
A standard thin brick (8 in x 2.25 in with a 3/8 in joint) needs about 7 bricks per square foot. Larger or smaller bricks change the count, so use the calculator for accuracy.
How much mortar do I need for thin brick veneer?
Plan on roughly 1 bag (80 lb) of Type S mortar per 50 thin bricks. Mortar covers both the setting bed and the grout joints between bricks.
How much does brick veneer cost per square foot?
Thin brick veneer costs $8-$15 per square foot installed, including bricks, mortar, and labor. Materials alone run $3-$6 per square foot.
