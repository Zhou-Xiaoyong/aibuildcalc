How to Use This Calculator

Our pipe size calculator follows UPC and IPC guidelines to recommend the right pipe diameter.

  1. Add up WSFU for all fixtures
  2. Measure static pressure at meter
  3. Determine longest pipe run
  4. Enter specifications
  5. Consider future additions

Frequently Asked Questions

What size pipe for a main water line?
Most homes use 3/4" or 1" copper/PEX for main supply. Size depends on fixtures and pressure.
What are WSFU?
Water Supply Fixture Units: bathroom faucet = 1, shower = 2, toilet = 2.5, kitchen sink = 1.5, washing machine = 4.
