How to Use This Calculator

Proper lighting design balances brightness, energy use, and comfort. Our calculator uses IES footcandle recommendations to size your fixture layout.

  1. Measure the room floor area in square feet
  2. Select the room type to set footcandle target
  3. Review total lumens, fixture count, and LED watts
  4. Space fixtures evenly for uniform illumination

Frequently Asked Questions

How many lumens do I need per square foot?
Living rooms need 10-20 lumens per sq ft, offices 30-40, kitchens 50-70, and garages 30-50. Multiply the room area by the footcandle requirement for total lumens.
How many LED watts equal a 60W incandescent bulb?
A 60W incandescent bulb produces about 800 lumens, which an LED achieves with only 9-10 watts. LEDs use roughly 1/6 the wattage for the same light output.
How many light fixtures do I need for a 200 sq ft kitchen?
A kitchen needs about 50 footcandles, so 200 sq ft requires 10,000 lumens. At 800 lumens per fixture, you would need 13 fixtures (ceiling lights or recessed cans).
