How to Use This Calculator

Building a wooden staircase requires precise measurements to ensure safety and code compliance. Our calculator helps you determine the number of steps, stringer length, and material needs.

  1. Measure the total vertical rise (floor to floor height)
  2. Choose a target riser height (7-7.75 inches is standard)
  3. Select a tread depth (10-11 inches is standard)
  4. Enter your stair width and number of stringers
  5. The calculator adjusts to the nearest whole step and recalculates actual riser height
Expert Insights Smart analysis for your stair project
📐
Rise + Run Rule
The sum of one riser and one tread should equal 17-18 inches for comfortable stairs. Our calculator flags combinations outside this range.
🏠
IRC Code Compliance
International Residential Code requires max 7.75" riser height and min 10" tread depth. Our calculator verifies compliance automatically.
🪵
Stringer Lumber
Always use 2x12 lumber for stringers. 2x10 boards do not leave enough wood after notching, creating a weak point at the throat.

Calculation Formula

Number of Steps = Total Rise / Desired Riser Height (rounded) Actual Riser Height = Total Rise / Number of Steps Total Run = (Number of Steps - 1) x Tread Depth Stringer Length = sqrt(Total Rise^2 + Total Run^2)

Frequently Asked Questions

What is the standard rise and run for stairs?
Standard rise is 7 to 7.75 inches per step. Standard run (tread depth) is 10 to 11 inches. The ideal sum of rise + run is 17-18 inches.
How many stringers do I need for a staircase?
For stairs 36 inches wide, use 3 stringers. For stairs 48 inches wide, use 4 stringers. Space stringers 16-18 inches on center.
What size lumber for stair stringers?
Use 2x12 lumber for stair stringers. Never use 2x10 or smaller, as the notches weaken the board significantly.
